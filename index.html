Restaurant Reviews

  • Technology utilized for the Restaurant Reviews project:

  • HTML5
  • CSS3
  • SASS
  • Javascript
  • Node JS
  • Babel
  • Webpack
  • Gulp
  • Git
  • Demo

July 2018 The Restaurant Reviews web application was created as the main project during the Grow with Google scholarship program for the Mobile Web Specialist Nanodegree. The application was written using Vanilla JavaScript without the use of frameworks or libraries. The primary focus of the project was to create a progressive web application which is offline-enabled through the use of Service Workers, IndexedDB, and the Cache API while maintaining accessibility standards and achieving high performance.

  • Restaurant Reviews project features:

  • Responsive Design
  • Progressive Web Application
  • Offline-enabled
  • Adheres to Accessibility Standards
